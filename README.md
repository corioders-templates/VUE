# Vue template by corioders

This is vue.js template used in our **[cli](https://github.com/corioders-templates/cli)**. It's created to easier start vue project. If you want to use vanilla js check our [template for vue](https://github.com/corioders-templates/VUE).

## Prerequisites

Before you begin, ensure you have met the following requirements:

- You have installed [node.js](https://nodejs.org/en/download/)
- You have installed [yarn](https://classic.yarnpkg.com/en/docs/install)
- You have installed [go](https://golang.org/dl/)
- **You have installed our [cli](https://github.com/corioders-templates/cli)**

## Setup

```bash
# create your project using our cli
corioders create

# you can install dependencies using our cli
corioders install
```

## Running project

```bash
# run your project using our cli
corioders run
```
